#### 5.1.4 总结

***解决哈希冲突：开方寻址法***
通过重新探测新位置的方法来解决哈希冲突

线性探查法： hash(key)+1, hash(key)+2, ......
二次探测法： hash(key)+1^2, hash(key)+2^2

***解决哈希冲突：链表法*** 
每个哈希槽对应一个链表

***解决装载因子过大问题***
* 预估容量，申请一个足够大的哈希表
* 动态扩容
